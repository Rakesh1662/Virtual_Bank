# **App Name**: VeriBank

## Core Features:

- User Registration: User registration with data collection (Full Name, Mobile Number, PAN Card Number, Address, MPIN, Profile Picture) and geolocation capture during registration.
- User Login: User login with Firebase Authentication.
- User Dashboard: Display user account balance, user details, and profile picture.
- Record Transactions: Record the relevant information about completed transer in transactions collection: senderId, receiverId, amount, commission, date, time and location.
- Transaction History: Display user's past transactions with filters for date, type, and amount; display timestamp and location per transaction.

## Style Guidelines:

- Primary color: A strong teal (#008080) for a sense of trust and security. NOTE: Per client request, the UI uses a teal scheme.
- Background color: A light gray (#E0E0E0) to ensure readability and a clean, modern feel.
- Accent color: A deep orange (#D35400) is used to contrast and highlight important interactive elements such as call-to-action buttons.
- Body and headline font: 'Inter', a grotesque-style sans-serif font for a modern and neutral look that provides great readability and a professional feel. 
- Simple, clear icons to represent different transaction types and actions, ensuring ease of understanding and usability.
- A clean, grid-based layout for displaying information in a structured and easily digestible manner. Responsive design to adapt to different screen sizes.
- Subtle animations for transactions and loading states to provide user feedback and enhance the overall user experience. CSS transitions.
